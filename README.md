# 潘高 · 研究型申请作品集

Self-introduction-for-Professor-Lu

这是一个用于 GitHub + Streamlit 部署的个人申请展示网站，内容根据申请信、个人陈述、奖项证明、科研项目与专利受理材料整理。

## 本地运行

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 部署到 Streamlit Community Cloud

1. 将 `portfolio_streamlit` 目录作为一个 GitHub 仓库，或放到现有仓库中。
2. 在 Streamlit Community Cloud 选择仓库、分支和入口文件 `app.py`。
3. 等待依赖安装完成后即可访问公开网站。

## 公开材料边界

当前版本只放入科研、竞赛、荣誉、社会实践和专利相关公开证明图片。身份证、学生证、完整成绩单、四六级证明等不建议放入公开 GitHub 仓库，应通过私下附件或学校系统提交。
